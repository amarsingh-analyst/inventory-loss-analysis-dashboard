# 📊 Inventory Loss Analysis Dashboard

Power BI dashboard analyzing inventory discrepancies and identifying missing stock across warehouse yards.

## 📌 Project Overview
This project analyzes inventory discrepancies such as missing, extra, and short items using Power BI.

## 🔧 Tools Used
- Power BI
- DAX
- Data Cleaning

## 📊 Key Features
- KPI cards for total missing items and quantity
- Pie chart for status distribution
- Bar chart for missing quantity by yard
- Interactive slicer for yard filtering

## 💡 Insights
- Identified highest loss yard
- Found distribution of missing vs extra vs short items

## 📁 Files
- PBIX dashboard file
- PDF dashboard preview

## 🚀 Outcome
Helps in reducing inventory loss and improving warehouse management decisions.
